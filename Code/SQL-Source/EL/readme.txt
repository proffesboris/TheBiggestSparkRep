Прототип для формирования связей по экономическим критериям (Операция EL) для K7M
Автор скриптов Ключко Павел.
Предназначено для запуска на ЛД.
Источники:
	1) ЕКС
	2) ФОК
	3) Интергрум
	4) K7M Базис в виде  t_team_cynomys.tka_client_info 
	5) K7M Проводки(выходная) в виде t_team_cynomys.yia_kras_010217


Описание скриптов (в порядке запуска):
1) stg_abs.sql заполнение таблиц 0 слоя и промежуточных агрегатов, которые не зависят от даты отчета.
2) stg_day_acc_rest.sql заполнение остатков на счетах ЗАВИСИТ от даты отчета. Параметр ${hiveconf:DT1} (Строка типа 2017-12-01)
3) stg_day.sql заполнение таблиц 0 слоя и промежуточных агрегатов, которые ЗАВИСЯТ от даты отчета. Параметр ${hiveconf:DT1} (Строка типа 2017-12-01)
4) EL5211.sql EL5212_5214.sql EL5213.sql EL5222.sql - расчет соответствующих критериев.ЗАВИСИТ от даты отчета. Параметр ${hiveconf:DT1} (Строка типа 2017-12-01)

"!ATTENTION_FILTER.txt" - фильтр по ИНН, которые надо применить после расчета (исключить ИНН, которые отобрать по алгоритму, который описан в файле)

all_EL_to_Nikita.sql - скрипт, который собирает представление для Никиты Зелинского

Для удобства прилагаются 2 shell-скрипта:
при проведении расчета сначала запустить stg_abs.sh,
а затем для расчета за  нужные даты запустить  stg_abs.sh, изменяя параметр  DT1 - сейчас установлена дата 2017-12-01

Результат для Никиты Зелинского накапливается в таблице  t_team_cynomys.k7m_EL
